# Glossary

## Artifact
A tangible output generated by an agent run, such as a chart (PNG), a data table (CSV), or a code block. Artifacts are intended for user consumption and are displayed in the Side Panel.

## Agent Graph
A directed cyclic graph (DCG) built with LangGraph that defines the state machine and logic steps of the AI agent.

## Checkpoint
A persistent snapshot of the agent's state at a particular point in time. Saved to PostgreSQL, allowing runs to be resumed or audited.

## Execution Trace
A sequential list of events (tool calls, node starts, results) that occurred during a specific agent run.

## Microsandbox Executor
The isolated service responsible for running arbitrary code generated by the agent.

## Presigned URL
A time-limited URL that allows a client (e.g., the Browser) to directly upload/download a file from MinIO without needing the master credentials.

## Thread
A persistent conversation container between a User and an Agent. A thread can contain multiple Messages and Runs.

## Workspace
The internal file system of the agent for a specific thread, where it saves and loads data files during its reasoning process.
