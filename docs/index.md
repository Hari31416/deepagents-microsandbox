# DeepAgent Sandbox Wiki

Welcome to the technical documentation for the **DeepAgent Sandbox**.

This project provides a robust environment for building, testing, and deploying data-focused AI agents. It integrates a FastAPI backend, a React frontend, and an isolated sandbox executor for safe code execution.

## Core Pillars

- **Safety & Isolation**: Code generated by agents runs in a dedicated `microsandbox-executor` container, ensuring the host system remains secure.
- **Traceability**: Every agent action, tool call, and state change is tracked and stored for debugging and audit.
- **Rich Interaction**: A modern React-based UI provides a seamless chat experience with real-time feedback and artifact management.
- **Session Controls**: Conversations can now be created, renamed, selected, and deleted directly from the sidebar.
- **Extensibility**: Built on top of LangGraph and the DeepAgents framework, allowing for flexible agent logic and tool integration.
